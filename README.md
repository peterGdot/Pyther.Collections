 Pyther.Collections

Provides tree & graph collections with various algorithms.

- **Graph** 
  - Directed Graph
  - Undirected Graph
- **Tree**
  - Binary Tree
  - Binary Min Heap
  - Binary Max Heap
- **Algoritms**
  - **Graph**
    - Bread-First Search
    - Depth-First Search
    - Shortest Path (Dijkstra)
    - **Directed Graph**
      - Topological Sort (Kahn's)
      - Topological Sort (DFS)
      - IsCyclic (Kahn's)
      - IsCyclic (DFS)
